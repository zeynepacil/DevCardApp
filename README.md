🚀 NexGen DevOS - Kişisel Gelişim ve Kariyer Yönetim Sistemi
NexGen DevOS, yazılım mühendisliği öğrencilerinin kendi gelişim süreçlerini oyunlaştırma (gamification) prensipleriyle takip etmelerini sağlayan bir mobil platformdur. Proje, teknik bilgi ölçümü, proje dağıtım simülasyonu ve yetenek ağacı yönetimi gibi özelliklerle donatılmıştır.

✨ Öne Çıkan Özellikler
🎭 Dinamik Geliştirici Kimliği: GitHub API entegrasyonu ile gerçek zamanlı Repo ve Takipçi sayılarını çeker. Kullanıcının XP seviyesine göre "Intern"den "Senior Architect"e kadar rütbe atamasını sağlar.

🧠 Akıllı Teknik Terminal: Yazılım dünyasından güncel ve kafa karıştırıcı sorular içeren quiz modülü. Doğru cevaplar XP ve TP (Tech Points) kazandırır.

⏱️ Odaklanma Sprintleri: Pomodoro benzeri sayaç sistemi ile verimli çalışma sürelerini ödüllendirir.

🏗️ Proje Hub & Deployment: Yeni projeler oluşturma ve "Deployment" simülasyonu. Başarıyla tamamlanan her proje kullanıcıya yüksek XP sağlar.

🌳 Yetenek Ağacı (Skill Tree): Kazanılan TP'ler ile mevcut yeteneklerin yüzdesel olarak geliştirilmesini sağlayan RPG mekaniği.

💾 Kalıcı Veri: AsyncStorage entegrasyonu ile tüm ilerleme cihaz hafızasında güvenle saklanır.

🛠️ Teknoloji Yığını
Framework: React Native (Expo)

State Yönetimi: Context API

Veri Saklama: AsyncStorage

API: GitHub REST API

UI/UX: Dark Mode / Terminal Aesthetic

🚀 Kurulum ve Çalıştırma
Projeyi yerel ortamınızda çalıştırmak için:

Bu depoyu clonlayın:
git clone https://github.com/zeynep/nexgen-devos.git

Proje dizinine gidin:
cd nexgen-devos

Gerekli paketleri yükleyin:
npm install

Uygulamayı başlatın:
npx expo start
